## 二、根因分析

### 2.1 描述行（`.ic-text`）

```
.ic-text (display:flex; align-items:center; gap:2px)
├── <ha-icon>  ← srv_icon, 14×14px
├── <span>text with badges</span>  ← 0.9em, 含 inline 角标
└── <img> + <span class="ic-qty">  ← 24×24 / 30×30
```

- **整体正确**：`align-items:center` 已把 ha-icon、文字、图片垂直居中
- **srv_icon 偏移**：ha-icon 作为 flex 子元素，在部分场景下 align-self 未显式设置，依赖继承，可能受嵌套容器影响
- **徽章偏移**：`ic-badge` / `ic-badge-num` 使用 `inline-block`/`inline-flex`，默认 `vertical-align:baseline`，其内部文字基线高于相邻文本基线 → 徽章整体偏上

### 2.2 图标组标题行

```
Block 模式：
.ig-title-row (display:flex; align-items:center; gap:8px)
└── <span style="white-space:nowrap;font-size:0.85em">
    ├── <ha-icon>  ← title_icon_html, 14×14px, vertical-align:middle（flex内无效）
    └── text

非 Block 模式：
.ig-title-row (display:flex; align-items:center; gap:8px)
├── <span class="ig-title-badge" style="font-size:0.65em">  ← align-items:baseline
│   ├── <ha-icon>  ← 14×14px
│   └── <span class="ig-badge-text">  ← top:1px
│       └── text
└── <hr class="ig-title-line">
```

- **错因**：`.ig-title-badge` 用 `align-items:baseline`，ha-icon 没有文字基线，其底部被对齐到文本基线 → ha-icon 偏高
- **补丁**：`top:1px` 把文字往下拉，但 ha-icon 仍偏高，视觉上只部分修正

### 2.3 本质矛盾

ha-icon / img / ic-badge 都是**替换元素或 inline-block**，没有文字基线。当它们与纯 text 放在同一个 flex/flow 容器内时：
- `align-items:baseline` → 替换元素底部 = 文字基线，替换元素会偏上
- `align-items:center` → 替换元素与文字互相对齐几何中心，视觉正确
- `vertical-align:baseline`（inline-block 默认）→ 同样问题，在非 flex 内联流中出现

### 2.4 徽章在非 flex 容器中的对齐

`ic-badge` / `ic-badge-num` 被 `_render_badges()` 生成为 HTML 内联元素，直接嵌入描述文本（`dtext_rendered`）和 ig 标题文本（`_ig_title_text`）。这些文本流**不是 flex 容器**，因此 `align-items:center` 不生效，徽章靠 `vertical-align` 决定垂直位置。当前未设置 `vertical-align`，默认 `baseline` 导致偏移。

### 2.5 复制键按钮内的对齐

`ic-copy-key` 是普通 `<button>`（非 flex 容器），内部并排 `<ha-icon>` + `<span>`：
- 按钮 `line-height:1.5`，ha-icon `--mdc-icon-size:13px`
- 两个子元素都是 inline 流，`vertical-align` 默认 `baseline`
- ha-icon 底部对齐文字基线 → 13px 图标在 1.5 倍行高的容器中偏上
- 与徽章同理，需 `vertical-align:middle`

## 三、统一方案

### 3.1 CSS 变更（`make_ic_css()`）

#### 删除
```css
/* 删除 ① */
ha-card .ig-badge-text{position:relative!important;top:1px!important}

/* 修改 ② — baseline → center */
ha-card .ig-title-badge{display:inline-flex!important;align-items:center!important;...}
```

#### 新增
```css
/* 确保 ig 标题行内 ha-icon 对齐 */
ha-card .ig-title-row ha-icon{flex-shrink:0!important;align-self:center!important}
ha-card .ig-title-badge ha-icon{flex-shrink:0!important}

/* 描述行内 ha-icon 统一对齐 */
ha-card .ic-text ha-icon{align-self:center!important}

/* 徽章与相邻文字垂直居中（非 flex 内联流） */
ha-card .ic-badge{vertical-align:middle!important}
ha-card .ic-badge-num{vertical-align:middle!important}

/* 复制键按钮内 ha-icon 与文字对齐 */
ha-card .ic-copy-key ha-icon{vertical-align:middle!important}
```

### 3.2 HTML 变更（ig 标题生成，约 L1904-1939）

#### 删除 `ig-badge-text` 包装器

```python
# 旧（非 block 模式）：
parts.append('<span class="ig-title-badge" style="white-space:nowrap;font-size:0.85em">{}<span class="ig-badge-text">{}</span></span>'.format(title_icon_html, _ig_title_text))

# 新：
parts.append('<span class="ig-title-badge" style="white-space:nowrap;font-size:0.85em">{}{}</span>'.format(title_icon_html, _ig_title_text))
```

#### 删除 inline `vertical-align:middle`

```python
# 旧：
title_icon_html = '<ha-icon icon="mdi:{}" style="color:inherit;--mdc-icon-size:14px;width:14px;height:14px;margin-right:2px;vertical-align:middle"></ha-icon>'

# 新（3 处统一）：
title_icon_html = '<ha-icon icon="mdi:{}" style="color:inherit;--mdc-icon-size:14px;width:14px;height:14px;margin-right:2px"></ha-icon>'
```

同样处理 linear_maps 和 block_maps 分支中的 ha-icon（L1917、L1921）。

### 3.3 描述行 CSS 微调

```css
/* L58 已有 align-items:center，只需确认 ha-icon align-self */
ha-card .info-card-block .ic-body .ic-text ha-icon{flex-shrink:0!important;align-self:center!important}
```

（当前 L59 已有 `flex-shrink:0`，只需追加 `align-self:center`；不涉及水平间距调整）
