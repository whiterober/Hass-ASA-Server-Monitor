# 对齐方案 v3：统一 center 对齐 + 图标比例匹配

> 核心思路：不换对齐方式，而是让图标大小匹配文字大小，使比例接近描述行（1.1:1）。

---

## 比例计算

| 场景 | 描述行（基准） | ig-title-badge | block 模式 |
|------|:---:|:---:|:---:|
| 文字字号 | 0.9em = 12.6px | **0.85em = 11.9px** | **0.85em = 11.9px** |
| 目标图标 | 14px | **12px** | **12px** |
| 图标:文字 | 1.11 | **1.01** | **1.01** |

copy-key：12.6px 文字 + 13px 图标 = 比例 1.03，合格，改容器为 inline-flex 即可。

---

## 变更清单

### CSS 变更（`make_ic_css()`）

#### 删除
```
ha-card .ig-title-row > span ha-icon{vertical-align:middle!important}  ← 不再需要
```

#### 修改
```
ha-card .ig-title-badge{...font-size:0.85em!important;line-height:1!important}  ← 对齐 block 模式字号
```

#### 新增
```css
/* 块模式 ig 标题：flex 容器，图标与文字自动 center */
ha-card .ig-title-row > span{display:inline-flex!important;align-items:center!important}

/* ig 标题徽章内图标缩小匹配 0.65em */
ha-card .ig-title-badge ha-icon{flex-shrink:0!important}
```

#### 按钮 CSS（`css +=` 块，约 L2467）
```css
ha-card .ic-copy-key{
  display:inline-flex!important;align-items:center!important;gap:4px!important;
  ...其他不变...
}
```

### HTML 变更（ig 标题生成）

#### 图标尺寸缩小（3 处 ha-icon + 删除 `vertical-align:middle`）

```python
# ig-title-badge（font-size:0.85em = 11.9px → icon 12px）
title_icon_html = '<ha-icon icon="mdi:{}" style="color:inherit;--mdc-icon-size:12px;width:12px;height:12px;margin-right:2px"></ha-icon>'

# linear_maps 分支同理 12px
# block_maps 分支同理 12px
```

#### 删除 `ig-badge-text` 包装器（已改，保持）
