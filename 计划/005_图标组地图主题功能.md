## 一、可复用组件（零新函数）

描述行的地图选择系统由以下组成，icon_group 全部直接复用：

| 函数 | ID 模式 | 作用 | icon_group 复用方式 |
|------|---------|------|---------------------|
| `toggleDescServerPopup(bi, di)` | — | 弹出地图选择浮窗 | 直接调用，不修改 |
| `selectDescServer(bi, di, sid)` | `mcICDescServer{bi}_{di}` / `mcICDescSrvStates{bi}_{di}` | 3态切换 + 存盘 | 共用相同 hidden input ID |
| `updateDescSrvBadge(bi, di, states)` | `mcICDescSrvBtn{bi}_{di}` | 更新按钮视觉 | 共用相同按钮 ID |
| `getServerBadge(sid)` | — | 渲染地图徽章 | 直接调用 |

> 关键：icon_group 行与文字描述行在同级 `descriptions[]` 中，`(bi, di)` 唯一不冲突。所有函数参数只有 `(bi, di)`，icon_group 和文字描述行共享同一套 hidden input ID。

## 二、编辑器 UI（asa-admin.html）

### 2.1 服务器按钮位置

按钮放在标题行右侧、删除按钮左侧，与描述行结构一致：

```
┌──────────────────────────────────────────────────┐
│ ⋮⋮  [标题______________________]  🌍  ✕        │
│     🖼️     🖼️     🖼️     ＋                    │
└──────────────────────────────────────────────────┘
```

> 注意：图标组没有 ×N 数量标记，只有纯图标。

新增 HTML（在标题行 `</span>` 和删除按钮之间插入）：
```html
<input type="hidden" id="mcICDescServer{bi}_{di}" value="{d.server}">
<input type="hidden" id="mcICDescSrvStates{bi}_{di}" value="{server_states JSON}">
<button class="btn-small btn-server" id="mcICDescSrvBtn{bi}_{di}"
  onclick="toggleDescServerPopup(this,{bi},{di})">
  {getServerBadge 或 active map icon}
</button>
```

### 2.2 addICIconGroup 初始化

新增行时 hidden input 初始化为空：
```
mcICDescServer{bi}_{di} = ""
mcICDescSrvStates{bi}_{di} = "{}"
```

## 三、保存逻辑（asa-admin.html — saveMCBlocks）

icon_group 行保存时追加 `server` 和 `server_states` 字段：

```javascript
b.descriptions.push({
  type: 'icon_group',
  title: igTitle,
  icons: igIcons,
  server: document.getElementById('mcICDescServer'+bi+'_'+igDi)?.value || '',

```

## 四、Lovelace 渲染（build_lovelace.py）

### 4.1 数据结构读取

与描述文字行完全一致的读取逻辑：
```python
dserver = desc.get('server', '')
srv_states = desc.get('server_states', {})
if not srv_states and dserver:
    srv_states = {dserver: 2}
linear_maps = [k for k,v in srv_states.items() if v == 1]
block_maps = [k for k,v in srv_states.items() if v == 2]
```

### 4.2 HTML 结构

用 `<div class="ig-row-wrapper ic-linear-Isl">` 包裹。标题行 = 左对齐标题 + 右侧分隔线（`hr flex:1`），图标行下方用 `::after` 全宽分隔线：

```html
<div class="ig-row-wrapper ic-linear-Isl">
  <!-- 标题行：左对齐标题 + 右侧分隔线填满剩余空间 -->
  <div class="ig-title-row" style="display:flex;align-items:center;gap:8px">
    <span class="ig-title-badge" style="white-space:nowrap;font-size:0.85em">
      <ha-icon icon="mdi:island"></ha-icon> 携带装备
    </span>
    <hr class="ig-title-line" style="flex:1;min-width:0" />
  </div>
  <!-- 图标行 -->
  <span class="ig-row">图标们</span>
</div>
```

### 4.3 三种渲染态

#### 无地图（默认）— 左对齐标题 + 同排分隔线 + 下方分隔线

```
携带装备 ──────────────────   ← 标题(左) + hr 填满右侧，万用色
🖼 🖼 🖼 🖼
────────────────────────────  ← ::after 整条分隔线，万用色，opacity 0.15
```

- 标题：`var(--primary-text-color)`，半透主题色底
- 同排分隔线：`border-top: 1px solid var(--primary-text-color)`，opacity 0.15
- 下方分隔线：CSS `::after`，同上色值

#### 线性态（state=1，如孤岛 🏝️）— 布局同默认，配色换地图色

```
🏝 携带装备 ──────────────────   ← ha-icon + 标题(左,孤岛橙) + hr(孤岛橙)
🖼 🖼 🖼 🖼
────────────────────────────    ← ::after 整条分隔线(孤岛橙)，opacity 0.4
```

- 标题文字颜色 = 地图色，背景 = 淡地图色 rgba 0.15
- ha-icon 颜色 = 地图色（复用 `.ic-linear-X .mdi` 现有 CSS）
- 同排分隔线 + 下方分隔线：地图色，opacity 0.4

#### 色块态（state=2，如孤岛 🏝️）

```
┌──────────────────────────────┐
│ ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓ │
│ 🏝 携带装备                  │  ← 白字
│ 🖼 🖼 🖼 🖼                  │
│ ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓ │
└──────────────────────────────┘
```

效果：整行地图色背景 + 白字 + 6px 圆角（复用 `.ic-text.ic-block-{sid}` 现有规则，CSS 零新增）

### 4.4 CSS 新增


## 🚫 禁区声明（强制）

| 禁区 | 原因 |
|------|------|
| **选图逻辑** | `openIconPicker_desc`、`selectDescImage`、`clearDescImageSelection`、`saveComboIG`、`toggleIconColorModeDesc`、`rebuildIGContainer` 等选图/反色/组合图函数**绝不允许修改** |
