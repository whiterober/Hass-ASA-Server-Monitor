## 二、🚨 关键差异（会导致两端渲染不一致）

> ⚠️ 2026-07-14 深入分析：preview_server.py L448-454 已通过 `parse_theme_yaml` 注入完整 HA 主题变量，**`var(--primary-text-color)` 在 preview 中可用**。差异 1/2/5 均可直接用 `color-mix` + `var()` 替代硬编码 `[data-theme]`。

### 差异 1：`_default` 服务器色块态背景

| 文件 | CSS 规则 | 机制 |
|------|----------|------|
| build | `color-mix(in srgb, var(--primary-text-color) 10%, transparent)!important` | HA 主题自适应 |
| preview | `[data-theme="dark"] rgba(255,255,255,0.1)` / `[data-theme="light"] rgba(0,0,0,0.1)` | 硬编码固定值（历史遗留） |

- **风险**：⚠️ → 🟢 低 — preview 已注入 `--primary-text-color`，**可直接改回 build 的 color-mix 规则**，删除 `[data-theme]` 双规则
- **收敛方向**：preview 直接替换为 `color-mix(in srgb, var(--primary-text-color) 10%, transparent)!important`

### 差异 2：badge-hollow（空心徽章）auto 模式

| 文件 | 选择器 | 值 |
|------|--------|-----|
| build | `[class*="ic-block-"][class*="ic-auto-block"] .ic-badge-hollow` | `color-mix(in srgb, var(--primary-text-color) 30%, transparent)` |
| preview | `[data-theme="dark"] .ic-auto-block .ic-badge-hollow` + `[data-theme="light"] ...` | `rgba(0,0,0,0.3)` / `rgba(255,255,255,0.3)` |

- **风险**：⚠️ → 🟢 低 — preview 有 `--primary-text-color`，**可直接改用 build 的选择器 + color-mix**，删除 `[data-theme]` 双规则

### 差异 3：`ic-desc-img` margin 值 + mixed_content 死代码（🔴 高优先级）

| 位置 | 值 |
|------|-----|
| build IC_CSS | `margin:0 1px 0 1px!important` |
| preview mixed_content 段（L233-268） | `margin:0 2px!important` |
| preview unconditional 段（L403-437） | `margin:0 1px!important`（与 build 一致） |

- **根因**：mixed_content 段（L233-268）的 ic-desc-img / ic-block-img / ic-qty / badge-hollow 规则**全部是死代码**——unconditional 段在所有类型之后无条件执行，覆盖 mixed_content 段的同属性规则
- **修复**：删除 preview L233-268 整段（~35 行死代码），保留 L270+ 的 block_type 判断。margin 自动统一为 `1px`

### 差异 4：icon_group 缺失规则

| 规则 | build | preview |
|------|:-----:|:-------:|
| `ha-card .ig-title-row ha-icon,ha-card .ig-title-badge ha-icon{color:inherit!important}` | ✅ 有 | ❌ 缺失 |

- **实际影响**：🟢 仅影响标题徽章内 ha-icon 从标题文字继承颜色。图标颜色来自 IC_CSS per-map 通用规则（`.ic-linear-{sid} .mdi`），不受此规则影响
- **修复**：preview 补一行 `css += 'ha-card .ig-title-row ha-icon,ha-card .ig-title-badge ha-icon{color:inherit!important}'`

### 差异 5：`_default` qty 描边主题适配策略

| 文件 | auto 模式 | 非 auto 模式 |
|------|-----------|-------------|
| build | 单条 `var(--primary-text-color)` | `var(--ic-block-bg)` |
| preview | `[data-theme]` 双规则（值相同） | 同 build |

- **风险**：⚠️ → 🟢 低 — preview 有 `--primary-text-color`，**可直接用 build 的单条规则**，删除 `[data-theme]` 冗余双规则
