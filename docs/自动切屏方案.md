## 原理

Windows 自带 `displayswitch.exe`（`C:\Windows\System32\`），就是 `Win+P` 弹窗背后的命令行工具：

| 参数 | 效果 | 对应 Win+P |
|------|------|-----------|
| `/internal` | 仅主屏幕（显示器） | 仅电脑屏幕 |
| `/external` | 仅第二屏幕（电视） | 仅第二屏幕 |
| `/extend` | 扩展 | 扩展 |
| `/clone` | 复制 | 复制 |

> 💡 工具位于 `B:\DisplaySwitch\DisplaySwitch.exe`，已验证有效（系统自带 `displayswitch.exe` 原理相同，作为后备）。
>
> ⚠️ **Steam Shell 验证**：游戏账号如果用了 Steam Shell（替换 Explorer 作为 Shell），需先用游戏账号手动跑一次 `B:\DisplaySwitch\DisplaySwitch.exe /external` 确认在该环境下能正常切屏。

## 三、办公账号自动切到显示器

### 计划任务（⚡ 最快 + 无黑框）

计划任务「At logon」触发器在用户会话建立时立即触发，比 `shell:startup` 和 Registry Run 都早。

#### 步骤 1：确认 VBS 已存在

`B:\DisplaySwitch\一键切换电脑.vbs` 已就绪（内容为 `/internal`），无需新建。

#### 步骤 2：创建计划任务

以**管理员身份**运行：

```powershell
schtasks /create /tn "SwitchToDisplay" /tr "wscript.exe //B //nologo B:\DisplaySwitch\一键切换电脑.vbs" /sc onlogon /it /rl highest /f
```

| 参数 | 含义 |
|------|------|
| `/sc onlogon` | 用户登录时触发（最早） |
| `/it` | 仅当前用户登录时运行 |
| `/rl highest` | 最高权限 |
| `/f` | 覆盖已有同名任务 |
| `//B` | wscript 静默模式，无弹窗 |

> 💡 之后每次登录办公账号，切屏几乎瞬间生效，完全无黑框。

## 四、游戏账号自动切到电视

> ✅ 已采用下方「五、Steam Shell 整合」方案，切屏集成在 `SteamShell.cmd` 中，效果完美。无需额外计划任务或启动文件夹。

### 实现方式

见 [五、与 Steam Shell 整合](#五与-steam-shell-整合推荐)。

---

## 五、与 Steam Shell 整合（推荐）

直接在 `SteamShell.cmd` 中集成切屏：

```batch
@echo off
B:\DisplaySwitch\DisplaySwitch.exe /external
start "" "C:\Program Files (x86)\Steam\steam.exe" -gamepadui
```

### 效果

```
人脸 → 登录 → 电视亮 → Steam Deck 界面
```

> 比放启动目录更丝滑，一步到位。

## 六、管理已创建的任务

```powershell
# 查看
schtasks /query /tn "SwitchToDisplay"

# 删除
schtasks /delete /tn "SwitchToDisplay" /f
```

---

## 七、保留手动切屏能力

`DisplaySwitch.exe` 只执行一次，不驻留、不锁屏幕。

任何时候仍可手动切换：

- `Win + P` 快捷键
- Windows 显示设置
- NVIDIA App

## 🏁 推荐最终结构

```
办公账号                               游戏账号
───────                               ───────
登录 → 计划任务 SwitchToDisplay        登录 → SteamShell（内置切屏）
  │   (wscript → 一键切换电脑.vbs)          │
  ▼                                      ▼
DisplaySwitch /internal                  SteamShell.cmd → /external → Steam
  │                                      │
  ▼                                      ▼
仅显示器亮（瞬间·无黑框）               仅电视亮 → Steam Gamepad UI ✅
```

> 这样最干净，也方便后面接入 HASS 的「电视模式 / 显示器模式」按钮。
