# 🖥️ 自动切屏方案 — Windows 双账号双屏自动切换

> 📅 2026-07-07 | 最稳、最简单、最符合现状的方案

---

## 🎯 目标流程

```
办公账号（指纹登录）                    游戏账号（人脸登录）
      │                                      │
      ▼                                      ▼
  自动切到显示器                          自动切到电视
                                              │
                                              ▼
                                         Steam Shell
```

### ✨ 优势

| | |
|---|---|
| ✅ 不影响手动切屏 | `Win + P` / 显示设置 / NVIDIA App 仍然可用 |
| ✅ 不依赖 HASS | 本地 Windows 启动即生效 |
| ✅ 不改显示器连接方式 | 纯软件方案 |
| ✅ 可后续接入 HASS | 保留远程控制扩展能力 |

---

## 一、下载 MultiMonitorTool

| 项目 | 内容 |
|------|------|
| 下载 | [NirSoft MultiMonitorTool](https://www.nirsoft.net/utils/multi_monitor_tool.html) |
| 解压到 | `C:\Tools\MultiMonitorTool\` |
| 确认 | 目录中有 `MultiMonitorTool.exe` |

---

## 二、创建两个显示配置

### 配置 A：电视模式

1. 手动调整 Windows 显示设置为：
   - **电视 = 主显示器**
   - 显示器 = 副显示器（或仅电视）
2. 以 **管理员身份** 打开 CMD，执行：

```batch
cd C:\Tools\MultiMonitorTool
MultiMonitorTool.exe /SaveConfig TV.cfg
```

> 生成 `TV.cfg`

---

### 配置 B：显示器模式

1. 手动调整 Windows 显示设置为：
   - **显示器 = 主显示器**
   - 电视 = 副显示器（或仅显示器）
2. 在管理员 CMD 中执行：

```batch
cd C:\Tools\MultiMonitorTool
MultiMonitorTool.exe /SaveConfig GAME.cfg
```

> 生成 `GAME.cfg`

---

### 📁 最终目录结构

```
C:\Tools\MultiMonitorTool\
├── MultiMonitorTool.exe
├── TV.cfg          ← 电视模式
└── GAME.cfg        ← 显示器模式
```

---

## 三、验证配置

### 切到电视

```powershell
C:\Tools\MultiMonitorTool\MultiMonitorTool.exe /LoadConfig C:\Tools\MultiMonitorTool\TV.cfg
```

### 切到显示器

```powershell
C:\Tools\MultiMonitorTool\MultiMonitorTool.exe /LoadConfig C:\Tools\MultiMonitorTool\GAME.cfg
```

> 确认两次切换均正常后继续。

---

## 四、办公账号自动加载 GAME.cfg

### 操作步骤

1. 登录 **办公账号**
2. `Win + R` → 输入 `shell:startup` → 进入启动文件夹
3. 创建 `GameMonitor.bat`：

```batch
@echo off
timeout /t 3 /nobreak > nul
C:\Tools\MultiMonitorTool\MultiMonitorTool.exe /LoadConfig C:\Tools\MultiMonitorTool\GAME.cfg
```

### 效果

```
办公账号登录 → 3 秒后 → 自动切到显示器模式
```

---

## 五、游戏账号自动加载 TV.cfg

### 操作步骤

1. 切换到 **游戏账号**
2. `Win + R` → 输入 `shell:startup`（⚠️ 这是游戏账号**自己的**启动目录）
3. 创建 `TVMonitor.bat`：

```batch
@echo off
timeout /t 3 /nobreak > nul
C:\Tools\MultiMonitorTool\MultiMonitorTool.exe /LoadConfig C:\Tools\MultiMonitorTool\TV.cfg
```

### 效果

```
人脸登录 → 3 秒后 → 切电视 → Steam Shell
```

---

## 六、与 Steam Shell 整合（推荐）

直接在 `SteamShell.cmd` 中集成切屏：

```batch
@echo off
C:\Tools\MultiMonitorTool\MultiMonitorTool.exe /LoadConfig C:\Tools\MultiMonitorTool\TV.cfg
timeout /t 2 /nobreak > nul
start "" "C:\Program Files (x86)\Steam\steam.exe" -gamepadui
```

### 效果

```
人脸 → 登录 → 电视切主屏 → Steam Deck 界面
```

> 比放启动目录更丝滑，一步到位。

---

## 七、保留手动切屏能力

`/LoadConfig` 只执行一次，**不会驻留、不会监听、不会锁屏幕**。

任何时候仍可手动切换：

- `Win + P` 快捷键
- Windows 显示设置
- NVIDIA App

---

## 🏁 推荐最终结构

```
办公账号                          游戏账号
───────                          ───────
Explorer                         SteamShell
  │                                │
  ▼                                ▼
GameMonitor.bat                  Load TV.cfg
  │                                │
  ▼                                ▼
显示器模式                        Steam Gamepad UI
```

> 这样最干净，也方便后面接入 HASS 的「电视模式 / 显示器模式」按钮。
