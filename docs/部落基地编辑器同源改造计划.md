# 部落基地编辑器同源改造计划

> 日期：2026-07-14 | 状态：🟡 待执行

## 目标

部落基地编辑器使用与服务器规则/部落运维**完全相同的代码路径**（`loadEditorData` → `renderMixedEditor` → `saveEditor`），零 copy，直接复用。

## 改造后调用链

```
导航点击 → loadEditorData(mode)
              │
              ├─ mode='server_rules' → EDITOR_CONFIGS.server_rules
              ├─ mode='tribe_ops'    → EDITOR_CONFIGS.tribe_ops
              └─ mode='base_ref'     → EDITOR_CONFIGS.base_ref  ← 新！
              │
              └─ loadJSON(cfg.file) → editorData
                   └─ renderMixedEditor(cfg.label)
                        │
                        ├─ cfg.renderHeader?.()     ← base: 服务器标签栏
                        ├─ grid-2 + 左TAB列表
                        ├─ cfg.renderLeftExtras?.() ← base: 复制自...
                        └─ 右面板(card+标题+blocks+保存)
                             └─ renderMixedContent(tab)

保存 → saveEditor()
         ├─ cfg.idPrefix → 读 DOM 名称/描述
         ├─ saveMCBlocks(tab)
         └─ saveJSON(cfg.file, editorData)
```

## 改动清单

### 改动 1：`renderMixedEditor` 加两个扩展点（+4 行）

在函数开头：
```javascript
let html = '';
if (cfg.renderHeader) html += cfg.renderHeader();
```

在 `+ TAB` 按钮后面：
```javascript
html += '<button ... onclick="addTab();reRender()">+ TAB</button>';
if (cfg.renderLeftExtras) html += cfg.renderLeftExtras();
```

### 改动 2：`EDITOR_CONFIGS` 加 `base_ref` 条目（+6 行）

```javascript
base_ref: {
  label: '部落基地',
  file: 'asa_base_quick_ref.json',
  idPrefix: 'brTab',
  saveFn: 'saveEditor()',
  renderHeader() {
    let h = '<div class="card" style="flex-shrink:0"><div class="server-tabs">';
    SERVER_LIST.forEach(s => {
      h += '<span class="server-tab'+(s.id===brSelectedServer?' active':'')+
        '" onclick="brSelectedServer=\''+s.id+'\';reRender();setTimeout(showPreview,200);">'+s.name+'</span>';
    });
    return h + '</div></div>';
  },
  renderLeftExtras() {
    return '<button class="btn-small" style="display:block;width:100%;margin-top:4px;padding:6px 0;border:1px solid var(--border);background:transparent;color:var(--text);opacity:0.75" onclick="copyBRFrom()">复制自...</button>';
  }
}
```

### 改动 3：导航栏改一行（±0 行）

```javascript
// 改前
else if (panel === 'base-ref') loadBaseRef();

// 改后
else if (panel === 'base-ref') loadEditorData('base_ref');
```

### 删除 1：`loadBaseRef()` 函数（-15 行）

整个函数不再需要 — `loadEditorData('base_ref')` 完全等价。

### 删除 2：`renderBaseRef()` 函数（-48 行）

整个函数不再需要 — `renderMixedEditor('部落基地')` 通过 `cfg.renderHeader` 生成服务器栏。
