# 展开折叠改造实施计划

> 日期：2026-07-19 | 方案2：JS onclick + CSS transition  
> 涉及：`build_lovelace.py` + `preview_server.py`

---

## 设计要点

1. **折叠标识**：右上角固定，折叠态低可视（`opacity:0.25`），展开态显眼（`opacity:0.7`）
2. **交互**：折叠态点整行→展开；展开态点标识→折叠（防误触）
3. **手风琴**：展开一个自动折叠同级其他
4. **不改原有折叠区域逻辑**：`ic-body` 内容结构不变

---

## 原型

```
折叠态（低可视标识）：
┌─────────────────────────────────┐
│ 🌾 采集倍率              6倍  ▸│  ← 整行可点
└─────────────────────────────────┘

展开态（标识变显眼）：
┌─────────────────────────────────┐
│ 🌾 采集倍率              6倍  ▼│  ← 点▼才折叠
│ 描述内容...                     │
│ 描述内容...                     │
└─────────────────────────────────┘
```

## 代码改动

### 1. HTML 结构改动（build_lovelace.py）

**改前：**
```html
<details class="ic-acc" name="ic-acc">
  <summary class="ic-summary">...</summary>
  <div class="ic-body">...</div>
</details>
```

**改后：**
```html
<div class="ic-acc collapsed" data-acc-group="ic-acc">
  <div class="ic-summary" onclick="toggleAcc(this)">...</div>
  <span class="ic-acc-arrow" onclick="toggleAcc(this.parentElement.querySelector('.ic-summary'))">▸</span>
  <div class="ic-body">...</div>
</div>
```

- `<details>` → `<div class="ic-acc collapsed">`
- `<summary>` → `<div class="ic-summary">`
- 新增 `<span class="ic-acc-arrow">` 右上角折叠标识
- `name="ic-acc"` → `data-acc-group="ic-acc"`（手风琴分组）

### 2. JS 手风琴逻辑（build_lovelace.py 内联）

```javascript
function toggleAcc(summary) {
  var acc = summary.parentElement;  // div.ic-acc
  var group = acc.getAttribute('data-acc-group');
  var isCollapsed = acc.classList.contains('collapsed');
  
  if (isCollapsed) {
    // 手风琴：折叠同级所有
    if (group) {
      var siblings = acc.parentElement.querySelectorAll('[data-acc-group="'+group+'"]');
      siblings.forEach(function(s) {
        if (s !== acc && !s.classList.contains('collapsed')) {
          collapseAcc(s);
        }
      });
    }
    // 展开当前
    expandAcc(acc);
  }
  // 折叠态：整行点击已触发 expandAcc
  // 展开态：arrow 点击才触发折叠
}
function expandAcc(acc) {
  acc.classList.remove('collapsed');
  var body = acc.querySelector('.ic-body');
  acc.style.maxHeight = (body.scrollHeight + 40) + 'px';  // 40=summary高度
}
function collapseAcc(acc) {
  acc.style.maxHeight = acc.scrollHeight + 'px';
  requestAnimationFrame(function() {
    acc.classList.add('collapsed');
    acc.style.maxHeight = '';
  });
}
// 箭头点击折叠
document.addEventListener('click', function(e) {
  if (e.target.classList.contains('ic-acc-arrow')) {
    var acc = e.target.parentElement;
    if (!acc.classList.contains('collapsed')) {
      collapseAcc(acc);
    }
  }
});
```

### 3. CSS 改动（make_ic_css）

```css
/* 折叠容器 */
ha-card .ic-acc { overflow:hidden; position:relative; }
ha-card .ic-acc:not(.collapsed) { max-height:none; }
ha-card .ic-acc.collapsed { max-height:28px; }
ha-card .ic-acc { transition:max-height 0.3s ease; }

/* 标题行可点击 */
ha-card .ic-summary { cursor:pointer; }

/* 折叠标识：右上角固定 */
ha-card .ic-acc-arrow {
  position:absolute; top:6px; right:8px;
  font-size:12px; color:var(--secondary-text-color);
  transition:opacity 0.2s, transform 0.2s;
  user-select:none; z-index:1;
}
/* 折叠态低可视 */
ha-card .ic-acc.collapsed .ic-acc-arrow { opacity:0.25; transform:rotate(0deg); }
/* 展开态显眼 */
ha-card .ic-acc:not(.collapsed) .ic-acc-arrow { opacity:0.7; transform:rotate(90deg); cursor:pointer; }
```

### 4. 手风琴 JS polyfill 移除

原有 Android WebView 的 `ontoggle` polyfill 可删除（方案2 JS 手风琴替代了 `details name` 属性）。
