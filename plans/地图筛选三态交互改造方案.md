# 地图筛选三态交互改造方案

> 创建：2026-06-25 | 状态：草案

---

## 需求重述

浮窗（`descServerPopup`）内每个地图按钮从**二态（选中/未选中）**改为**三态循环**：

| 状态 | 视觉效果（按钮） | 保存后预览效果 |
|------|-----------------|-------------|
| 0 — 未选中 | 透明度更低（比现在更淡） | 无 |
| 1 — 线性选中 | 完全不透明，图标线性高亮 | 图标着色为地图主题色 |
| 2 — 色块选中 | 带背景色圆角块（≈现有） | 描述行 → 淡淡主题色圆角框背景 |

点击循环：未选中 → 线性选中 → 色块选中 → 未选中 → …

---

## 涉及文件

| 文件 | 改动范围 | 复杂度 |
|------|---------|--------|
| `www/asa-admin.html` | popup 渲染 + 点击处理 + 数据存取 | 🔴 高 |
| `build_lovelace.py` | 预览渲染：描述行根据 `server_state` 渲染不同样式 | 🟡 中 |
| `preview_server.py` | 新增三态预览 CSS | 🟢 低 |

## 数据结构变更

### 当前（二态）

```json
// 描述行
{ "text": "经验倍率 6x", "server": "Abe" }
// server 为空 → 不限地图；非空 → 仅该地图
```

### 目标（三态 per-map）

```json
// 描述行
{ 
  "text": "经验倍率 6x", 
  "server_states": { "Abe": 2, "Cen": 1, "Isl": 0 }
}
// server_states: 0=未选中, 1=线性, 2=色块
// 空 map → 视为 server_states={}
// 无对应 key 的地图 → 默认 0
```

> **向后兼容**：旧数据 `"server": "Abe"` 自动映射为 `"server_states": {"Abe": 2}`（色块=旧默认行为）

## 详细改动清单

### 一、`asa-admin.html` — 管理后台

#### 1.1 数据存储改造（`~L2840` saveMixedEditor）

**当前**：
```js
// 读取 mcICDescServer{bi}_{di}.value → 单值
desc_data.server = document.getElementById('mcICDescServer'+bi+'_'+origDi)?.value || '';
```

**改为**：
```js
// 读取隐藏的多值字段
const statesStr = document.getElementById('mcICDescSrvStates'+bi+'_'+origDi)?.value || '{}';
desc_data.server_states = JSON.parse(statesStr);
// 删除旧 server 字段
delete desc_data.server;
```

#### 1.2 隐藏字段改造

新增隐藏 input：`mcICDescSrvStates{bi}_{di}`，值格式：`'{"Abe":2,"Cen":1}'`

#### 1.3 popup 渲染改造（`~L5358` toggleDescServerPopup）

每个服务器按钮从单选改为三态独立按钮：

```js
// 读取当前 states
const curStates = JSON.parse(hiddenInput.value || '{}');
const curState = curStates[k] || 0; // 0/1/2

// 渲染按钮
btn.style.opacity = curState === 0 ? '0.35' : '1';
if (curState === 1) {
  // 线性选中：无背景，图标着色
  btn.style.background = 'transparent';
  btn.style.color = SERVER_MAP[k].c;
  btn.style.border = '1px solid ' + SERVER_MAP[k].c;
} else if (curState === 2) {
  // 色块选中：现有行为
  btn.style.background = SERVER_MAP[k].c;
  btn.style.color = '#fff';
  btn.style.border = 'none';
}
```

#### 1.4 点击处理改造（`~L5391` selectDescServer）

```js
function selectDescServer(bi, di, sid) {
  const hiddenEl = document.getElementById('mcICDescSrvStates'+bi+'_'+di);
  let states = JSON.parse(hiddenEl.value || '{}');
  
  // 循环：0→1→2→0
  states[sid] = ((states[sid] || 0) + 1) % 3;
  if (states[sid] === 0) delete states[sid]; // 清理0值
  
  hiddenEl.value = JSON.stringify(states);
  updateServerBadge(bi, di, states); // 更新按钮徽章
  closePopup();
}
```

#### 1.5 色块/线性计数徽章

按钮旁的 count-badge 显示：`线性:N 色块:M`

#### 1.6 读取旧数据兼容

渲染描述行时：
```js
if (desc.server && !desc.server_states) {
  desc.server_states = { [desc.server]: 2 }; // 迁移旧单值
}
```

### 二、`build_lovelace.py` — 预览渲染

#### 2.1 数据读取改造

`render_info_card()` 描述行渲染（约 L2103）：

```python
# 当前
dserver = desc.get('server', '')
dserver_attr = f' data-server="{dserver}"' if dserver else ' data-server=""'

# 改为
srv_states = desc.get('server_states', {})
# 向后兼容
if not srv_states and desc.get('server'):
    srv_states = {desc['server']: 2}
# 渲染 data-server-states
states_json = json.dumps(srv_states)
dserver_attr = f' data-server-states=\'{states_json}\''
```

#### 2.2 描述行 CSS class

根据 states 最高级别决定：
- 全是 0 → 无特殊 class
- 有 1（线性）→ 图标着色 CSS（如 `.desc-linear-Abe { color: #9c27b0; }`）
- 有 2（色块）→ 背景框 CSS（如 `.desc-block-Abe { background: rgba(156,39,176,0.12); border-radius: 6px; }`）

需要在动态 CSS 中为每个 `active_map` 生成对应规则。

#### 2.3 filter_js 改造

```js
// 当前：显示/隐藏
// 改为：三态显示
// state=0 → display:none
// state=1 → 图标着色
// state=2 → 背景框
```

#### 2.4 map_filter 区块改造

按钮栏中的每个地图按钮也需要展示三态：
- 默认全部为"未选中"状态
- 点击一个 → 该地图变为"线性"或"色块"（取决于需要）

> 注：map_filter 区块的三态可能与 popup 不同。需用户确认 map_filter 预览栏的交互逻辑。
