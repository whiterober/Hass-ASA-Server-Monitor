# 📖 项目名词表（统一说法）

> 本表为 `.github` 下所有指令、SKILL、规则文件的统一术语参考。
> 所有新增或修改 .github 文件时，必须对齐本表说法。

---

## 🔤 会话结构

| 术语 | 英文 | 定义 |
|------|------|------|
| **会话** | session | 从打开聊天窗口到当前的全部对话 |
| **回合** | turn | 单次一问一答（用户发一条 → 智能体回复一次） |
| **上下文** | context | LLM 的上下文窗口/长度（技术指标，非"会话"） |

> ⚠️ 区分：说"本次对话"用「会话」，说"上下文太长"用「上下文」。

---

## 📊 报告流程

| 术语 | 定义 |
|------|------|
| **前置提醒** | 每回合第一步输出的状态行 `📋 进✓ 安✗ 省✓(...)` |
| **结束报告** | 回合末的 6 步完整报告（结论→小白→写进度→模型线→脚本→结算监控） |
| **结论** | 1-3 句概括本回合完成工作 |
| **小白解释** | 1 句非技术语言说明 |
| **进度写盘** | 将进度写入 session memory |
| **模型输出** | `📋 进✓/✗ 安✗/✓ 省(...) 省 ~¥X.XXX` 行 |
| **结算监控** | 对比脚本输出与前几回合的异常检测 |

---

## 💰 Token 与成本

| 术语 | 定义 |
|------|------|
| **R 规则** | token-audit SKILL 定义的省 token 规则（R1-R9） |
| **占用%** | 当前 token 使用量占总上限的百分比 |
| **缓存%** | 模型缓存命中率（衡量上下文复用效率） |
| **本回合费** | 当前回合消耗的费用估算（¥） |
| **Token 审计** | 每回合自动评估 token 风险与成本趋势 |

## 🛡️ 安全

| 术语 | 定义 |
|------|------|
| **安全审批** | 高风险操作前的强制确认流程 |
| **风险分级** | 低/中/高/严重 四级 |
| **Dry Run** | 预演：展示将要执行的操作但不实际执行 |
| **改前备份** | 修改文件前 Copy-Item 到 bak/ |

## 🎮 ASA 项目专用

| 术语 | 定义 |
|------|------|
| **实页** | HA 服务器上实际运行的 lovelace 页面 |
| **预览区/预览页** | ASA 后台右侧 iframe 实时预览（点「💾 保存Tab」自动刷新）。渲染效果以此为准，不跳到 HA lovelace 实页。 |
| **双端同步** | 改动同时应用到实页和预览区 |
| **ASA 后台** | ASA 服务器规则/部落运维/部落基地管理后台 |
| **lovelace** | Home Assistant 仪表板系统 |
| **卡片** | info_card 块类型（原"静态卡片"） |
| **静态卡片** | → 已改名为「卡片」 |

## 📝 进度追踪

| 术语 | 定义 |
|------|------|
| **进度文件** | 记录每回合完成工作的 markdown 文件 |
| **锚点** | 指向当前活跃进度文件的引用 |
| **尾追加** | 新进度直接追加到文件末尾 |
| **零读取优先** | 默认不读取进度文件内容，直接追加 |
